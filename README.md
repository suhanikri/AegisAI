# 🛡 AegisAI

**Behavioral Threat Intelligence System for AI-Driven Scam Detection**

---

## 🚨 Problem

With the rapid rise of AI-generated scams, deepfake voice fraud, emotional phishing, and authority impersonation attacks, traditional cybersecurity tools fail to detect psychological manipulation patterns.

Current systems detect malicious code —  
but not malicious intent.

---

## 💡 Solution

AegisAI introduces a **Behavioral Threat Intelligence Framework** that detects emotionally manipulative and AI-driven fraud signals across text and voice channels.

---

## 🔍 Key Features

- Emotional Coercion Index (ECI)
- Hybrid Risk Fusion Model
- Threat Category Classification
- Dynamic Risk Scoring
- Animated Risk Visualization
- Citizen Protection Mode
- Institutional Monitoring Mode
- Simulated Audio Threat Analysis

---

## 🏗 Architecture Overview

User Input (Text / Audio)
↓  
Behavioral Threat Engine  
↓  
Risk Fusion Layer  
↓  
Dashboard Output / Institutional Analytics

---

## 🛠 Tech Stack

- FastAPI (Backend)
- HTML/CSS/JavaScript (Frontend)
- TextBlob (Sentiment Analysis)
